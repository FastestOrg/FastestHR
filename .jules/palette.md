## 2024-05-01 - Add Password Visibility Toggle to ResetPassword
**Learning:** For authentication-related forms (such as `ResetPassword`), password and confirm password fields are often lacking a visibility toggle, causing poor UX and potential typos since the user cannot easily verify their complex password against strict rules.
**Action:** When working on password inputs, always add an integrated `Eye`/`EyeOff` toggle button to allow users to temporarily see their typed input.
